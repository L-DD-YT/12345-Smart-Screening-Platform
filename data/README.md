# 示例数据

本目录存放演示、导入测试用的脱敏/合成样本数据，**不包含真实个人信息**。

## `samples/` 目录说明

| 文件 | 说明 |
|------|------|
| `供参考线索_房山500条.csv` / `.xlsx` | 房山区域合成工单样本（批次 1） |
| `供参考线索_房山500条_批次2.csv` / `.xlsx` | 房山区域合成工单样本（批次 2，类别更丰富） |
| `模拟数据_房山1000条.xlsx` | 更大规模的演示导入样本 |
| `供参考线索.xls` / `供参考线索1.xls` | 早期参考表格格式样本 |

可在平台「线索汇集 → 表格导入」中上传 `.xls` / `.xlsx` 文件进行筛查演示。

## 重新生成样本

```bash
py scripts/generate_fangshan_500.py --batch 1 --count 500
py scripts/generate_fangshan_500.py --batch 2 --count 500
```

生成结果会写入 `data/samples/`。
